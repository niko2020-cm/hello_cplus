#pointForParameter函数的作用是用于理解，指针做函数参数的情况
两个浮点数进行比较的时候，由于浮点是是近似存储，不可以直接比较大小，要用两个数相减，如果小于一个充分小的数，那么就是相等的。

#constPointForParameter函数的作用是用于理解常指针做参数的情况
常指针做函数本身的情况,可以使用到传值的效率，又不会破坏原有的值（用于只读的情况）
